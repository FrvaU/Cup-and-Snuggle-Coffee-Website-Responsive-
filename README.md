# Cup-and-Snuggle-Coffee-Website-Responsive-
Basic Coffee Website using HTML and CSS
Overview:
Cup & Snuggle is a warm, inviting, and fully responsive coffee shop website designed to showcase the brand's cozy personality and delicious menu offerings. The website features a modern aesthetic with a soothing color palette inspired by coffee tones — cream, sand, caramel, and espresso.
The site is built with pure HTML, CSS, and JavaScript (no frameworks), making it lightweight, fast, and easy to customize. It includes a fully functional mobile navigation menu, smooth scrolling, and a visually appealing layout that works seamlessly across all devices.
Purpose:
The website serves as a digital storefront for Cup & Snuggle, a coffee shop with locations across Pakistan (Karachi, Islamabad, Lahore, Multan). Its goals are to:
Introduce the brand's cozy, welcoming atmosphere
Showcase featured drinks (Mocha, Hot Chocolate, Cappuccino, Latte)
Provide easy access to the full menu
Display location and contact information
Encourage online orders and in-store visits
Features:
Desktop & Mobile Responsive
Fully responsive design using CSS Grid and Flexbox
Mobile-first approach with breakpoints at 900px and 600px
Hamburger menu for mobile devices
Modern Visual Design
Warm, coffee-inspired color palette
Custom buttons with hover effects
Smooth card animations (lift effect on hover)
Glass-morphism header with backdrop blur
Mobile Navigation
Toggleable hamburger menu
Smooth open/close animation
Prevents body scroll when menu is open
Visual Content
Hero section with high-quality coffee imagery
Floating badge (4 cities across Pakistan)
Feature cards and menu cards with images
Interactive Elements
Call-to-action buttons (See Menu, Find a Location, Order Online)
Anchor links for smooth scrolling
Footer with location links, social media, and company info
Technologies Used
Technology	Purpose
HTML5	- Semantic structure
CSS3 - Styling, animations, responsive layout
Google Fonts -	Playfair Display + Lato fonts
